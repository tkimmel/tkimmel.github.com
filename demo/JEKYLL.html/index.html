What is Jekyll?
Static site generator
Why do we care?
CMS are generally for non-technical users
They require maintenance
they require thinking about performance
Jekyll serves a static site
you can use markup to blog quickly
you can drop in HTML / CSS / JS when you wanna be fancy
this is blogging for developers

you may need Ruby-ish things - we’ll cut to Aaron briefly if you do ‘cause he’s 100x the programmer I am :D

Get yourself a repo up and running: [GITHUB-HANDLE].github.com

cd USERNAME.github.com
git remote set-url origin git@github.com:USERNAME/USERNAME.github.com.git
git push origin master
get Jekyll on your local machine
gem install jekyll
go into your blog directory (the one you cloned) andjekyll serve
