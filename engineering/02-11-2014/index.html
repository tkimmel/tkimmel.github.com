VIN testing with Polk
Conclusion: 99% accuracy
Let’s look at how we got there, because it’s important:
We built a web scraper to go out and harvest VIN data from instavin.com/vehicles/*/*
This job ran for a while and yielded a list of 2.8 Million VINs (and we didn’t get IP blocked!)
We then randomly selected one thousand VINs from this list
Polk ran the list and found data for 990 out of 1000 VINs
NET: Internal random testing yielded a success rate of 99% thru Polk
If there is more partner testing to be done along similar lines, let us know
